Description: All about this Kirby Blog Theme.

Text: This theme is made to give you a nice and clean start to create a blog with Kirby. It allows you to setup your posts and pages quickly, so you can customize and just start blogging. Here are some build-in features of this theme in the current version:

- categories
- responsiveness
- link posts
- error page
- RSS feed for your articles
- pagination for the article overview
- static tweet button
- prearranged blueprints for (link: https://github.com/bastianallgeier/kirbycms-panel text: Kirby Panel) usage

##Kirby?
(link: http://getkirby.com text: Kirby) is a lightweight, flexible and file-based CMS made by (link: http://bastianallgeier.com text: Bastian Allgeier).

##How to install the theme
After downloading just remove the default **assets**, **content** and **site** folder and replace it with the downloaded ones. For optional interface usage you have to install the [Kirby Panel](https://github.com/bastianallgeier/kirbycms-panel/).

##Docs
If you want to learn more about Kirby, visit the (link: http://getkirby.com text: Kirby-Website) and check out the docs!
